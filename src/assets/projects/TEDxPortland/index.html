About

This site was designed and built for the several teams of the Managing Creativity class in Summer 2015 taught by Shannon Kaiser at the Art Institute of Portland as a student run project. Each team comes with a theme idea with a bold idea and statement to setup a TEDxPortland campaign for the year 2016. Each team has to come up with various ideas and content for the campaign following the TEDx guidelines and rules.

Deliverables

  • Theme
  • Logo
  • Speaker Packet
  • Social Media
  • SWAG
  • Etc...

Production

Site construction started started with creating a framework to house and display each groups' theme content. A DropBox was created to share content created within each team and to automaticly update content for me to utilize.

Problem solving became a crucial point in finding glitches and display problems by code. Some of the gallery display problems were caused by a simple missing file that failed to upload to the website server. Another key issue is to plan an architectual analysis of the project to set and find the content requirements ahead of time. Another part is to plan out the time needs and schedule with content and tools utilized in the site I want to use that fit in the time constraints with other projects and the same time.
